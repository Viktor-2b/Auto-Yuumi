# 🐱 全自动辅助系统 (auto yuumi)

为抵抗排位赛限制，避免无意义的匹配对战，研发的基于 Python 的集成屏幕监测、视觉校准与键鼠模拟的视觉自动化脚本。

---

## ⚠️ 核心前置要求

### 1. 必须以【管理员身份】运行

* **开发时：** 请右键点击 PyCharm 图标 -> `以管理员身份运行`。
* **部署时：** 若通过终端运行，请使用管理员权限打开 CMD 或 PowerShell。

### 2. 安装 Tesseract-OCR 引擎

1. 前往 [Tesseract 开源镜像站](https://github.com/UB-Mannheim/tesseract/wiki) 下载安装包。
2. 建议安装到默认路径：`C:\Program Files\Tesseract-OCR\`。
3. 如果修改了安装路径，请同步修改主代码中的 `pytesseract.tesseract_cmd` 变量路径。

### 3. 游戏内视频与 UI 设置

* **分辨率：** 必须设置为 `1024 x 768`。
* **窗口模式：** 必须设置为 **窗口化 (Windowed)**。
* **缩放：** 用户界面缩放和小地图比率为`50`。
* *注：只要游戏内分辨率正确，电脑桌面分辨率和缩放比例不会影响脚本运行。*

---

## 📦 环境配置与安装

1. **获取源码：** 克隆或下载本项目到一个纯净的文件夹中。
2. **虚拟环境：** 推荐使用 Python `venv` 或 `conda` 创建一个干净的 Python 3.8+ 虚拟环境。
3. **安装依赖库：** 在终端中运行以下命令，安装项目所需的第三方视觉与系统控制库：
```bash
pip install -r requirements.txt

```

## 🚀 运行方式

1. 在管理员终端中执行：
   ```bash
   python auto_yuumi.py
   ```
2. 打开英雄联盟客户端，配置好 LeagueAkari，开启匹配。
3. 脚本启动后会打印：`🤖 悠米专属高级自动化脚本已启动...`。随后可以去忙别的事情，脚本会自动接管接下来的所有流程。
